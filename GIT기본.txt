- 폴더 초기화
git init
초기화

- 깃허브 설치후 1회만 실행해도 됨
{
	git config --global user.name "이름"
	git config --global user.email "깃허브 메일"
}

git remote add origin http://
저장소 연결

git remote rm origin
저장소 삭제


- 풀과정
git pull origin master
모든자료를 끌어와 현재 내용과 병합



- 푸시과정
git status
로컬폴더와 깃의 싱크상태를 체크

git add .
변경된 파일을 모두 추적

git commit -m "메시지"
변겨사항 코멘트 안하면 푸시안됨

git push origin master
푸시
